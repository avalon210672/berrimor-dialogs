# Инструкция: Подключение собственного домена к проекту на Lovable

> 📅 Версия: 1.0
> 🎯 Назначение: Пошаговое руководство по подключению кастомного домена к проекту, размещённому на Lovable

## 🔍 Предварительные условия

Перед началом убедитесь, что:

- [ ] Проект создан и работает на Lovable (доступен по адресу `*.lovable.app`)
- [ ] Есть **активная платная подписка Lovable** (функция кастомных доменов доступна только на платных тарифах)
- [ ] Проект синхронизирован с GitHub (код в безопасности)
- [ ] Вы готовы потратить от 15 минут до 24 часов на распространение DNS-записей

---

## 📋 Вариант 1: Покупка домена напрямую через Lovable (рекомендуется для новичков)

Самый простой способ — Lovable всё настроит автоматически.

### Шаг 1. Перейдите в настройки публикации
1. Откройте ваш проект в Lovable
2. Нажмите кнопку **Publish** в правом верхнем углу

### Шаг 2. Выберите покупку домена
1. В разделе **Custom domain** нажмите **"Buy domain"**
2. Введите желаемое имя домена в поисковую строку
3. Просмотрите доступные варианты и зоны (.com, .ru, .app, .io и т.д.)
4. Выберите подходящий домен и добавьте в корзину

### Шаг 3. Оформите покупку
1. Заполните данные для регистрации (если требуется)
2. Оплатите домен банковской картой
3. Дождитесь подтверждения оплаты

### Шаг 4. Автоматическое подключение
1. После оплаты домен **автоматически подключится** к проекту
2. Lovable самостоятельно настроит DNS и выдаст SSL-сертификат
3. Статус изменится на **"Live"** в течение нескольких минут

### ✅ Результат
Ваш проект доступен по адресу: `https://ваш-домен.рф`

## 📋 Вариант 2: Подключение существующего домена (если домен уже куплен у регистратора)

Этот вариант требует ручной настройки DNS у вашего регистратора.

### Шаг 1. Подготовьте информацию о домене
Убедитесь, что:
- Домен зарегистрирован на ваше имя
- У вас есть доступ к панели управления DNS вашего регистратора (например, reg.ru, Namecheap, GoDaddy, Timeweb)

### Шаг 2. Начните подключение в Lovable
1. Откройте проект → кнопка **Publish**
2. В разделе **Custom domain** нажмите **"Connect domain"**
3. Введите ваш домен (например: `mydomain.com` или `мойпроект.рф`)
4. Нажмите **"Continue"**

### Шаг 3. Получите DNS-записи от Lovable
Lovable покажет список DNS-записей, которые нужно добавить. Обычно это:

| Тип записи | Имя | Значение |
| :--- | :--- | :--- |
| **A** | `@` (или `мойдомен.рф`) | `75.2.60.5` |
| **CNAME** | `www` | `custom-domains.lovable.app` |
| **TXT** (для верификации) | `_verification.мойдомен.рф` | `lovable-domain-verification=...` |

> ⚠️ **Важно**: Точные значения могут отличаться — используйте те, которые показаны в интерфейсе Lovable для вашего проекта!

### Шаг 4. Добавьте записи у регистратора
1. Войдите в панель управления вашего регистратора доменов
2. Найдите раздел **"Управление DNS"**, **"DNS-записи"** или **"Zone Editor"**
3. Добавьте каждую запись из таблицы выше:
   - **A-запись** для основного домена
   - **CNAME-запись** для поддомена `www`
   - **TXT-запись** для верификации (если требуется)

### Шаг 5. Подтвердите в Lovable
1. Вернитесь в Lovable
2. Нажмите кнопку **"Verify"**
3. Lovable начнёт проверку наличия записей
4. Статус изменится на **"Pending"** — это нормально

### Шаг 6. Дождитесь применения DNS
- DNS-записи могут распространяться от **нескольких минут до 48 часов**
- Обычно это занимает 1-2 часа
- Как только записи применятся, статус автоматически станет **"Live"**

### ✅ Результат
Ваш проект доступен по адресу: `https://ваш-домен.рф`

## 🎯 Что делать с несколькими проектами

### Оптимальная стратегия для вашего случая

**Если у вас несколько проектов и один оплаченный аккаунт Lovable:**

1. **Для главного проекта** (основной источник дохода):
   - Купите отдельный домен (например, `мойосновнойпроект.рф`)
   - Подключите его по инструкции выше

2. **Для остальных проектов** (портфолио, демо, тестовые):
   - **Вариант А**: Оставьте на поддоменах `*.lovable.app` — бесплатно и не требует действий
   - **Вариант Б**: Купите **один домен** и используйте поддомены:
     1. Купите, например, `моипроекты.рф`
     2. Подключите его к Lovable с **wildcard-записью** (CNAME `*` → `custom-domains.lovable.app`)
     3. Подключайте поддомены как `проект1.моипроекты.рф`, `проект2.моипроекты.рф` и т.д.

## 🔒 Проверка безопасности

После подключения домена обязательно проверьте:

### ✅ SSL-сертификат
- Откройте сайт по `https://ваш-домен.рф`
- В браузере должен быть зелёный замочек
- Lovable выдаёт бесплатный SSL автоматически

### ✅ Редирект с HTTP на HTTPS
- Попробуйте открыть `http://ваш-домен.рф`
- Должен произойти автоматический редирект на `https://`

### ✅ Работоспособность всех функций
- Проверьте авторизацию (если есть)
- Проверьте отправку форм
- Проверьте API-запросы к Supabase или другим сервисам

### ✅ Мобильная версия
- Откройте сайт на телефоне
- Убедитесь, что адаптивная вёрстка работает

### ✅ Старый домен
- Поддомен `*.lovable.app` продолжит работать
- При желании можно настроить редирект (в настройках Lovable)

## 🚨 Возможные проблемы и их решение

### Проблема 1: Долго не наступает статус "Live"
**Решение:**
- Проверьте, точно ли вы добавили все записи у регистратора
- Убедитесь, что нет опечаток в значениях
- Подождите ещё 2-3 часа — DNS может кешироваться провайдерами

### Проблема 2: Сайт открывается, но без HTTPS
**Решение:**
- Подождите до 15 минут — Lovable генерирует SSL автоматически после применения DNS
- Если прошёл час, обратитесь в поддержку Lovable

### Проблема 3: Не работает `www.ваш-домен.рф`
**Решение:**
- Проверьте, добавили ли вы CNAME-запись для `www`
- Убедитесь, что значение CNAME указано верно

### Проблема 4: Ошибка CORS или проблемы с авторизацией
**Решение:**
- Проверьте настройки Supabase (если используете) — добавьте новый домен в список разрешённых
- В некоторых API нужно добавить домен в белый список
